# Roshan_pandey.github.io — personal portfolio

This is a minimal static portfolio site ready for GitHub Pages.

How to use
1. (Optional) Customize the files:
   - Edit `index.html` for your name, bio, project links and email.
   - Replace the avatar element or swap it for an `<img>` in the hero section.
   - Update `styles.css` for color / typography.
2. Create the repository on GitHub named `Roshan_pandey.github.io`.
3. Push these files to the repo root and enable Pages.

Quick push (run from within the project folder)
```bash
git init
git add .
git commit -m "Initial site"
git branch -M main
git remote add origin https://github.com/Roshan_pandey/Roshan_pandey.github.io.git
git push -u origin main
```

Enable GitHub Pages
- For a user site (username.github.io) pushing to main branch root usually publishes automatically.
- Check Settings → Pages → Source: main (root). Save if needed.
- Your site will be available at: https://Roshan_pandey.github.io/
